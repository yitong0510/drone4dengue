# 📦 Drone4Dengue System Modules

## 1. Authentication & User Account

This module controls account creation, login, password management, and user-specific settings. It ensures secure access and user identification.

### Includes Use Cases:

* UC1: Login Account
* UC2: Register Account
* UC3: Reset Password
* UC4: Edit Profile
* UC11: Manage Settings

---

## 2. Drone & Surveillance Management

Handles registration and tracking of drones and their surveillance data. Admins can assign drones to areas and monitor image captures.

### Includes Use Cases:

* UC5: Manage Drone and Location
* UC6: Manage Images Captured by Drone

---

## 3. User Management & Access Control

Administer user roles, permissions, and registration status across the platform. Controls verification, updates, and access restrictions.

### Includes Use Cases:

* UC7: Manage User

---

## 4. Dengue Data Analytics

Supports uploading, filtering, and analyzing dengue case and weather data. It includes tools for exporting reports and visualizing outbreak trends.

### Includes Use Cases:

* UC8: Manage Dengue Data
* UC9: Generate Report
* UC14: Manage Weather Data

---

## 5. Prediction & Alert Management

Processes prediction data and manages alert rules, notification thresholds, and alert logs to inform users about risk areas.

### Includes Use Cases:

* UC10: Manage Prediction and Alert
* UC12: Get Potential Dengue Notification

---

## 6. Public Awareness & Prevention

Provides end users with recommendations based on risk levels and supports dengue awareness through real-time notifications.

### Includes Use Cases:

* UC12: Get Potential Dengue Notification
* UC13: Get Recommendations
* UC15: Monitor Daily Dengue Cases
* UC16: Create Dengue Location Alert
