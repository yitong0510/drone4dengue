# 🧭 UI Navigation Overview – Drone4Dengue

This guide summarizes the navigation structure for both the **mobile app** and **admin web dashboard**.

---

## 📱 DroneEye Mobile App Navigation

### 1. Dashboard (Home)

* Live risk level
* Latest alerts
* Navigation to: Notifications, Drone, Profile

### 2. Notifications

* Risk alerts (UC12)
* Tap to view detailed map & suggested actions

### 3. Recommendations

* Risk categories: High, Medium, Low (UC13)
* Tap for preventive tips

### 4. Profile

* View/edit profile (UC4)
* Settings: Notification preferences (UC11)

### 5. Drone

* Register/View personal drones (UC5)
* Status of monitoring coverage

### Bottom Navigation Menu:

`Dashboard | Recommendation | Notifications | Profile`

---

## 💻 Admin Web Dashboard Navigation

### Sidebar Menu:

* **Dashboard**: Risk map & stats (UC10)
* **User Management**: Add/Edit/Delete Users (UC7)
* **Drone Management**:

  * Drone list, add/edit/remove (UC5)
  * Image gallery (UC6)
* **Data Management**:

  * Upload dengue cases (UC8)
  * Upload weather data (UC14)
* **Reports**:

  * Filter & generate reports (UC9)
* **Settings**:

  * Profile, password, threshold, alerts (UC11)

---

## Notes:

* Mobile App: Bottom navigation bar
* Admin Web: Sidebar for module switching, top header for profile/logout
